== DELETE OBJECT

Delete an object.

== Description

Delete an object in the device.

== Shell example

Delete asymmetric key 0x52b6:

  yubihsm> delete 0 0x52b6 asymmetric

== Protocol details

=== Command

|===============
|T~c~ = 0x58
|L~c~ = 2 + 1
|V~c~ = I \|\| T
|===============

I := Object ID (2 bytes)

T := link:../Concepts/Object.html[Type] (1 byte)
