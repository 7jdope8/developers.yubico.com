== GET OPAQUE

Fetch an opaque object from device.

== Description

Retrieve an opaque data object (like a X.509 certificate) from the device.

== Shell example

Fetch opaque object 0xe255 and store in file cert.der:

  yubihsm> get opaque 0 0xe255 cert.der

== Protocol details

=== Command

|============
|T~c~ = 0x43
|L~c~ = 2
|V~c~ = K
|============

K := Key ID (2 bytes)

=== Response

|===========
|T~r~ = 0xc3
|L~r~ = L~D~
|V~r~ = D
|===========

D := Opaque data
