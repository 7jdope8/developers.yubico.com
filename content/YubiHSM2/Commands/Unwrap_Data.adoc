== UNWRAP DATA

Decrypt data

== Description

Using a wrapkey decrypt data.

== Shell example

  yubihsm> decrypt aesccm 0 0x5b3a MRkj6B0AAAAAAAAAAoO4dkIeAYoPvwTV/M/JX1dwKnLqnERO1hSW4wPS
  Hello world!

== Protocol details

=== Command

|===============
|T~c~ = 0x69
|L~c~ = 2 + 13 + L~D~ + 16
|V~c~ = K \|\| N \|\| D \|\| M
|===============

K := Key ID of a wrapping key (2 bytes)

N := Nonce (13 bytes)

D := Data to be unwrapped

M := Mac (16 bytes)

==== Response

|===========
|T~r~ = 0xe9
|L~r~ = L~D~
|V~r~ = D
|===========

D := Unwrapped data
