== CLOSE SESSION

Close session.

== Description

Thise closes the current session and releases it for re-use.

== Shell example

Close session 0:

  yubihsm> session close 0

== Protocol details

=== Command

|===========
|T~c~ = 0x40
|L~c~ = 0
|V~c~ = Ø
|===========

=== Response

|===========
|T~r~ = 0xc0
|L~r~ = 0
|V~r~ = Ø
|===========
